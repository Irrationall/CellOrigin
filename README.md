# CellOrigin

&nbsp;
  
## Prerequisites
* [minepy ver.1.2.6](https://github.com/minepy/minepy)
* numpy < 2.0 (Currently, mienpy doesn't work with numpy 2.0 or higher version)


&nbsp;

## Installation
git clone https://github.com/Irrationall/CellOrigin

cd CellOrigin

pip install -e .
